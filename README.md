# FlowPilot SaaS Landing Demo

Next.js 14, TypeScript ve Tailwind CSS ile hazırlanmış tek sayfalık SaaS landing demo sitesi.

## Komutlar

```bash
npm install
npm run dev
npm run lint
npm run build
npm run deploy:pages
npm run deploy:worker
```

`npm run build` statik çıktıyı `out/` klasörüne üretir.
